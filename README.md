# Hack & Juice

## Description

Hack & Juice est un projet Epitech de cybersécurité sous forme de CTF (Capture The Flag).
L'objectif est d'explorer et d'analyser une application web afin d'identifier des vulnérabilités
de sécurité, de les exploiter dans un cadre pédagogique, et de les documenter.

---

## Objectifs

- Identifier les failles de sécurité d'une application web
- Appliquer des techniques d'analyse et d'exploitation (OWASP, DevTools, etc.)
- Documenter chaque challenge avec méthodologie, vulnérabilités, risques et recommandations
- Développer une approche professionnelle de la cybersécurité offensive et défensive

---

## Méthodologie générale

Chaque challenge suit la structure de documentation suivante :

1. **Methodology** — étapes de découverte et d'exploitation
2. **Vulnerabilities** — failles identifiées avec composant affecté et sévérité
3. **Risks** — impacts potentiels de la vulnérabilité
4. **Actions** — stratégies de mitigation, correctifs et bonnes pratiques

---

## Outils utilisés

- **Browser DevTools** — inspection du code source, réseau, stockage
- **KeePass** — gestion et analyse de bases de données de mots de passe
- **OWASP Guidelines** — référentiel de classification des vulnérabilités
- **BurpSuite** — interception et analyse des requêtes web pour détecter des vulnérabilités et tester la sécurité des applications web
